# Raspberry Pi Compute Module 4 (CM4) 스펙

## 1. 기본 사양

| 항목 | 사양 |
|------|------|
| SoC | Broadcom BCM2711, Quad-core Cortex-A72 (ARMv8) 64-bit @ 1.5GHz |
| RAM | 1GB / 2GB / 4GB / 8GB LPDDR4-3200 SDRAM |
| eMMC | Lite (없음, SD 사용) / 8GB / 16GB / 32GB |
| 무선 | 옵션: Wi-Fi 802.11 b/g/n/ac (2.4/5GHz) + Bluetooth 5.0 BLE |
| 안테나 | PCB 내장 또는 U.FL 외부 안테나 커넥터 |
| 크기 | 55 x 40 mm |
| 커넥터 | Hirose DF40C-100DS-0.4V x 2 (200핀 메자닌) |
| 동작 온도 | 표준: -20 ~ +85°C / 산업용(2025~): -40 ~ +85°C |

## 2. 전기적 특성

| 항목 | 값 |
|------|-----|
| 입력 전압 | 5V (VBAT 핀) |
| 대기 소비전류 | ~400mA (2.0W) |
| 부하 소비전류 | ~1.4A (7.0W) |
| 3.3V 출력 | 최대 600mA |
| 1.8V 출력 | 최대 600mA |
| GPIO 전압 | 3.3V 또는 1.8V (GPIO_VREF 핀으로 선택) |

## 3. 인터페이스

| 인터페이스 | 사양 |
|-----------|------|
| Ethernet | Gigabit (BCM54210PE PHY 내장, PoE 옵션) |
| USB | USB 2.0 x 1 (호스트) |
| PCIe | Gen 2 x1 |
| HDMI | 듀얼 micro HDMI (4Kp60) |
| CSI | 듀얼 MIPI CSI-2 카메라 |
| DSI | 듀얼 MIPI DSI 디스플레이 |
| UART | 최대 6개 (UART0~5, PL011 + mini UART) |
| SPI | SPI0, SPI1 |
| I2C | I2C0, I2C1 |
| GPIO | 최대 28개 범용 GPIO |
| 비디오 디코딩 | H.265 4Kp60 / H.264 1080p60 |

## 4. eMMC 모델 선택 가이드

| 용도 | 권장 모델 |
|------|----------|
| RTU (최소) | CM4 Lite 2GB (SD카드 사용) |
| RTU (권장) | CM4 8GB eMMC + 2GB RAM (무선 불필요) |
| 개발용 | CM4 32GB eMMC + 4GB RAM + 무선 |
